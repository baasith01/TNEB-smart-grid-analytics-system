# TNEB Smart Grid Analytics System

## Project Overview

An end-to-end data analytics project that simulates a smart electricity grid management system for Tamil Nadu Electricity Board (TNEB).

The project integrates synthetic electricity consumption data with real Tamil Nadu weather data to generate actionable business insights.

## Objectives

- Analyze electricity consumption patterns
- Monitor transformer health
- Forecast future electricity demand
- Integrate weather data with energy usage
- Build executive dashboards
